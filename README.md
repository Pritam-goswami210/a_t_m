# a_t_m
# 🏦 Simple Python ATM Simulator
This is my very first Python project! I am a 14-year-old student (Class 9) aspiring to become an AI Research Engineer. 

## 🌟 Features
- **Smart Pincode Finder:** Uses logic to provide pincodes for Indian state capitals, including seasonal logic for Srinagar.
- **Banking Operations:** Allows users to Deposit and Withdraw money.
- **Input Validation:** Handles unknown commands using `if-elif-else` logic.

## 🚀 What I Learned
- How to use Python variables and user inputs.
- Improving code efficiency using `elif` instead of multiple `if` statements.
- Basic arithmetic operations for banking logic.

## 🛠️ How to Run
1. Download the `atm.py` file.
2. Run it using `python atm.py` in your terminal.
